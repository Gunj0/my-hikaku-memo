## main タグの高さ調整

```html
<!-- min-h-screen で最小の高さをスクリーンに合わせる -->
<body class="min-h-screen flex flex-col">
  <!-- h-16 で高さ固定 -->
  <header class="h-16 bg-gray-200">Header</header>
  <!-- flex-1 で残りの高さを埋める -->
  <main class="flex-1 bg-white">Main</main>
  <!-- h-12 で高さ固定 -->
  <footer class="h-12 bg-gray-200">Footer</footer>
</body>
```
