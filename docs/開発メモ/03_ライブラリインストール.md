## アップデート

- `pnpm outdated` で古いライブラリをチェック
- `pnpm update` でアップデート
  - `next`、`eslint-config-next`は最新にすると Cloudflare Worker が対応してない場合があるので注意
  - [OpenNext:AWS](https://opennext.js.org/aws/compatibility)
- `pnpm run preview` で動作確認
