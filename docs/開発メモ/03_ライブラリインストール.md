## アップデート

- メジャーバージョンのみ指定にして `pnpm update`

```json
  "dependencies": {
    "@opennextjs/cloudflare": "^1",
    "next": "15",
    "react": "19",
    "react-dom": "19"
  },
  "devDependencies": {
    "@eslint/eslintrc": "^3",
    "@tailwindcss/postcss": "^4",
    "@types/node": "^25",
    "@types/react": "^19",
    "@types/react-dom": "^19",
    "eslint": "^9",
    "eslint-config-next": "15",
    "tailwindcss": "^4",
    "typescript": "^5",
    "wrangler": "^4"
```

- `pnpm outdated` で古いライブラリをチェック
- バージョンを修正して、`pnpm update`
  - `next`、`eslint-config-next`は最新にすると Cloudflare Worker が対応してない場合があるので注意
- `pnpm run preview` で最後に起動チェック
